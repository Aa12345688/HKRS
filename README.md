# 皇凱貿易 - HKRS 庫存管理系統 (Industrial PURE Edition)

這是一個專為「機車改裝零件」量身打造的工業級庫存管理系統。本系統遵循「極度去繁就簡」的原則，移除冗餘的視覺干擾，轉化為一具冷靜、準確且高效的作業工具。

---

## 🛠️ 核心功能 (HKRS Core)

### 📂 唯一作業入口 (The Single Entry)
*   **掃瞄中心**：App 開啟即進入掃描模式。沒有儀表板、沒有複雜報表，專注於解決「進貨」與「出貨」的實務流程。
*   **黑白鮮明 UI**：採用高對比度的工業風格，確保在昏暗或強光環境下的倉庫中依然清晰。

### 🔊 語音同步反饋 (Audio Assistant)
*   **即時語音報品**：掃描成功後，系統會自動播報零件名稱與操作狀態。
*   **盲掃支援**：技師無需頻繁查看螢幕即可確認操作是否成功，大幅提升作業速度。

### 🖨️ 專業標籤列印 (Bulk Label Printing)
*   **批量標籤生成**：支援一鍵生成多個零件的一維條碼與 QR Code。
*   **A4 列印優化**：預設排版適合標準 A4 標籤紙，連結數位系統與實體貨架的唯一橋樑。

### 🔢 純粹數據管理 (Pure Data)
*   **去視覺預警**：移除所有「紅/綠/黃」庫存水位顏色警示，將管理判斷權回歸專業人員，系統僅誠實反映即時庫存數據。

## 🚀 快速啟動 (Setup)

### 1. 安裝依賴
```powershell
npm install
```

### 2. 本地開發
```powershell
npm run dev
```

---

## 🛠️ 技術棧 (Tech Stack)

*   **前端核心**: React 18 + Vite 7
*   **狀態管理**: Zustand (持久化於 LocalStorage)
*   **樣式系統**: TailwindCSS + Framer Motion (Industrial Style)
*   **音訊處理**: Web Speech API
*   **掃描技術**: html5-qrcode
